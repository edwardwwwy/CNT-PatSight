# Full-text raw layer

This directory is generated by `scripts/fetch_fulltext/`.

- `pdf/` and `html/` contain validated remote artifacts.
- `text/` contains faithful text derivatives produced by the parser.
- `reports/` contains acquisition run summaries.
- `fulltext.sqlite3` is the authoritative local acquisition registry.
- `fulltext_acquisition_queue.csv` is the derived A/B-only download queue.
- `fulltext_source.csv` and `fulltext_coverage.csv` are review-friendly exports.

Downloaded and derived content is ignored by Git because it may be large or
licensed. Local source PDFs under `data/raw/papers/` are referenced in place and
must never be overwritten by this layer.
