# B 类数据随机抽样质量审计（10 篇）

审计日期：2026-07-20  
抽样种子：`20260720-B-QA-v1`  
审计对象：B 类成果中已进入八表交付的 93 篇论文  
审计样本：10 篇论文、59 个成果 run、64 条工艺记录、270 条证据记录

## 结论

本次样本的核心实验事实通常可以回溯到论文，尤其是论文中以表格明确给出的温度、流量、产率、转化率和部分表征数值，抄录质量总体较好。但数据尚不满足“无需人工复核即可作为正式/金标准数据使用”的要求。

样本判定为：完全通过 0 篇，限定通过 4 篇，需修正 5 篇，重大修正 1 篇。最常见的系统性问题是把论文未报告的常压写成精确的 `101.325 kPa`：涉及 6/10 篇、40/64 条工艺记录。另发现跨 run/跨样品字段污染、单位字段违规、同一条件冲突值被静默合并、实验系列漏项，以及证据定位不能支撑完整记录等问题。

因此，B 类成果目前适合做“带原文回查的候选数据”，不宜直接晋升为正式库、金标准或模型训练标签。优先修复本报告列出的高风险规则后，再做一轮自动扫描和复抽检。

## 抽样与核查方法

总体限定为 `source_coverage.csv` 中 `eight_table_included=yes` 的 93 篇，而不是全部 371 篇已登记 B 类文献。对每个 `source_id` 计算：

```text
SHA256("20260720-B-QA-v1|<source_id>")
```

按哈希升序抽取前 10 篇。该方法固定、可复现，不使用标题、年份或成果状态参与挑选。

对抽中论文逐篇阅读全文：8 篇以 PDF 原页和关键图表渲染页核查，2 篇以完整 HTML 正文核查；随后交叉检查各论文目录下的八张 CSV。核查重点包括实验边界、条件与单位、结果数值、跨 run 归属、缺失实验、证据定位和现有问题日志覆盖度。

判定口径：

- **限定通过**：核心事实可靠，但存在局部、可规则化修复的问题，使用时仍需保留限制说明。
- **需修正**：有可能改变筛选、统计或复现实验结论的数据错误，修复前不宜正式使用。
- **重大修正**：存在实验漏项或失败 run 被写成成功结果等会反转语义的问题。

## 逐篇结论

### 1. LIT_0C298FD36DDDBFF2 — 需修正

论文中 Mn 含量系列、PP/催化剂用量、500/800 °C 两段温度、TPO 分解出的丝状碳/无定形碳含量、直径区间和 Raman 趋势均能与数据对应，核心数值质量较好。

主要问题是论文没有报告反应为常压，10 条工艺阶段却全部写成 `atmospheric` 和 `101.325 kPa`，并标成 reported；此外，原文“almost no carbon”被写成精确 `0 wt%`。前者会直接影响按压力筛选和复现实验。

### 2. LIT_29053831AED49C65 — 限定通过

PECVD 的 Co/Ni 催化层、C2H2/NH3 比例、偏压、时间及主要温度—压力窗口与论文一致，显式示例的阵列形貌和长度也基本正确。

相图边界中的部分点被实体化为精确 run，却仍标为 reported，未区分图上读取/近似值；`750 K` 被换算成 `476.85 °C`，产生不必要的两位小数且丢失原始单位。代表性 TEM 结论还被扩散到所有成功 run。核心概念可用，但应修正值状态和适用范围。

### 3. LIT_70C62F9DC9DF53B4 — 需修正

尾气组成、前后炉温、流量、FCNT/RCNT 产率、RCNT 长度和力学/电学性能数值均可在论文中找到。

但 FCNT 行错误继承了只属于 RCNT 的“约 20 cm”长度和水活化条件；RCNT 又被标为 `fresh catalyst`，与论文所述失活/回收铁颗粒相反。水的共进料流量还被塞入碳源流量字段。论文未报告常压，2 条记录却写成精确常压。属于明确的跨 run 污染。

### 4. LIT_66DB00E917C24EAC — 需修正

复合材料制备步骤和 M-pCNT 复合物的 Young 模量、拉伸强度、断裂应变、弯曲强度等关键数值与论文一致。

问题在于多个样品行都继承了 pCNT 与商业 CNT 混合后的直径、长度和残余 Fe 描述，无法按样品可靠筛选；证据统一指向结论页，但多数表征事实实际位于正文图表页。该论文研究的是 CNT/环氧复合材料，八表中的 `catalyst_system`、`carbon_source` 和 `yield_quality` 被语义重载，且大部分定量机械性能仅埋在自由文本中。

### 5. LIT_0B337E3E2AB8B340 — 需修正

反应时间、温度和乙炔流量三组实验的产率、BET 和多数直径值抄录准确，且共享的 600 °C/45 min/200 sccm 条件被去重。

然而同一名义条件在不同实验系列中给出 `17.5 nm` 和 `13 nm` 两个 CNT 直径，孔结构数值也有差异；成果静默保留其中一个值，没有记录冲突及实验系列来源。9 条工艺记录还全部补入论文未报告的精确常压。成本/安全判断被标为 reported，证据定位只有宽泛的 local span。

### 6. LIT_CA678F825E54BE45 — 需修正

论文确实研究常压热等离子体连续合成，钢带基材、功率、移动速度、常用乙烯/氢气流量及 MWCNT/包封 Fe 的总体现象可以回溯。

关键单位错误是把字符串 `1000-1300 K` 写入 `temperature_setpoint_C`；同时没有结构化论文结果中更具体的 1050–1070 K 有利区间。产率/产品证据指向装置图附近的文字，而真正的 MWCNT、TEM/TGA/EDX 结果位于后文。常用模拟配方也被绑定为一个精确实验 run，来源关联不充分。

### 7. LIT_308D1442E521B6DD — 限定通过

Fe 掺杂氧化铝基底、82/18 mol% H2/CH4、1000 °C、5 °C/min、无保温、25–30 nm 催化颗粒及 Raman/RBM 信息与论文相符；现有日志也正确提醒 Raman 推导直径只代表可见子集。

主要问题仍是论文未报告常压，却写成 `101.325 kPa`。删除该推断并区分 reported 与 normalized 后，可限定使用。

### 8. LIT_436402DA7A57D194 — 限定通过

流化床小试及 5.3/16 cm 放大实验的进料、时间、产物质量、转化率和 `X` 值与论文表格高度一致，是样本中显式表格抄录较好的一篇。

问题包括：CNT 生长压力未报告却统一写成常压；所有工艺表记录都标为 `lab_batch`，与成本表中的 bench/pilot 分级矛盾；由 `X/时间` 计算的生产率仍标为 reported。修复这些状态和分级问题后可用。

### 9. LIT_BD24584C41DAE4DD — 限定通过

8 nm Fe、750 °C、约 150 mbar、Ar/H2/C2H2 流量及三类催化剂相演化路径与论文一致，且将结果标为重复实验组而非受控独立 run 是合理的。

`~150 mbar` 被无保留地写成精确 `15 kPa`；论文给出的 γ-Fe 示例直径（平均 26 nm、标准差 11 nm、9–50 nm）未入表；各组的形貌和产物结论主要用摘要证据支撑，证据范围不足。可在保留近似性和组级边界后使用。

### 10. LIT_87B8240D8CDFE77C — 重大修正

论文表 2 给出了 9 个成功碳源的逐项产率、高度、G/D、密度和部分直径，并另有 methane 失败实验。成果只保留 8 个成功碳源，完整漏掉 p-xylene；同时 methane 失败行却继承了 `single-wall forest`、`vertically aligned forest`、直径和密度等成功结果。其余成功 run 也没有保存表 2 的逐碳源精确值，只保留全局区间。

这会把失败实验误判为成功并改变碳源排序，必须重建该论文全部 10 个 feed campaign 后再使用。论文同样没有明确报告常压，不应写成精确 `101.325 kPa`。

## 系统性问题与整改优先级

1. **先阻断未经来源支持的默认值。** 禁止把空压力自动写成 `atmospheric/101.325 kPa`；单位换算必须保存原始值、原始单位、近似符号和 `normalized/calculated` 状态。
2. **建立 run 级字段继承白名单。** 只有装置级、全系列明确共享的条件才能复制；长度、直径、形貌、活化条件、催化剂状态等结果字段默认不得跨 run 继承。
3. **保留实验系列与冲突值。** 同一名义条件在不同表/系列出现不同结果时，不能静默去重，应保留 series ID、原始值和冲突标志。
4. **做完整性对账。** 用论文表格/图例中的实验成员数与 `source_run` 行数核对，优先识别 p-xylene 这类漏项和 methane 这类失败 run 污染。
5. **强化证据粒度。** 每个关键字段应能定位到实际支持它的页、表、图或正文段落；摘要或结论不能替代详细形貌、尺寸和样品级数值的证据。
6. **分离非 CNT 合成任务。** 复合材料性能不应强行占用 CNT 合成催化剂/碳源/产率字段；建议使用独立 track/schema，或至少加入明确的语义映射层。
